Lynbrook First Tech Challenge

In Lynbrook FTC, we teach you the fundamentals of building a robot. From computer modeling to programming to hardware, we provide real world experience in robotics.

Learn More!

The Game

Every year, First releases a new game challenge, and we must build a robot that can solve the tasks at hand within two to three months. This year's challenge is Velocity Vortex!
Watch the Game
The basis of this game revolves around shooting plastic balls into raised game capsules and to develop a robot that can elevate beyond a certain height. As always, we will need to design a robot that can run with preprogrammed (autonomous) tasks and driver-controlled tasks.

speaker_phone

Hardware

In addition to tetrix pieces, we use custom 3D printed materials to design a fully functional robot. The entire robot can be run by an android phone attached directly to the robot that communicates with the driver.

code

Software

Through FTC, you'll learn how to make android applications with the Java programming language. Not only that, we will be developing autonomous programs that use infared sensors and teleop programs that function based on user controls.

group

Workshops

We provide our members with opportunities to learn how to use Android Studio, Inventor, and gain background in the electrical aspect of robotics through our workshops. We also mentor FLL teams and have created FTC youtube tutorials.

Why First Tech Challenge?

FTC offers a rewarding experience of receiving hands on experience in engineering. With the small team sizes that allow members to gain exposure in many different fields and the many opportunities for torunaments, FTC combines innovation, engineering, and fun.
